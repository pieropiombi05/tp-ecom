# E-Commerce Store Documentation

## Project Structure

- **/src**: Contains all the source code for the application, organized into components, services, and utilities.
- **/public**: Holds static assets such as images, stylesheets, and JavaScript files.
- **/tests**: Contains unit and integration tests for the application.
- **/config**: Configuration files for different environments (development, testing, production).

## Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/pieropiombi05/tp-ecom.git
   cd tp-ecom
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Create a `.env` file in the root directory and configure the required variables.
   
4. Start the application:
   ```bash
   npm start
   ```

## Usage Instructions

- Navigate to `http://localhost:3000` in your browser to access the application.
- Use the navigation menu to browse different clothing categories.
- Add items to your cart and proceed to checkout for purchasing.

For more information, refer to the project's [wiki](https://github.com/pieropiombi05/tp-ecom/wiki) or contact the development team.
